Core Travel Personality Dimensions
Travel Personality Vector (LLM Interpretation Guide)

The following values represent a user's travel personality profile.

Each dimension is a score between 0 and 1.

Interpretation:

0.0–0.3 → strong preference for the left trait

0.4–0.6 → balanced / situational

0.7–1.0 → strong preference for the right trait

These are soft behavioral tendencies, not strict rules.

When generating travel suggestions:

align strongly with high scoring traits

avoid strong conflicts with low scoring traits

balance across multiple dimensions

Dimensions

exploration_tolerance
0 = prefers familiar destinations
1 = seeks novelty, unusual places, unfamiliar cultures

structure_preference
0 = spontaneous, minimal planning
1 = structured itineraries and reservations

social_energy
0 = prefers solitude or quiet environments
1 = enjoys social interaction, events, meeting people

activity_intensity
0 = slow travel, relaxation, scenic wandering
1 = physically active travel, adventure, dense schedules

cultural_curiosity
0 = leisure and entertainment focus
1 = strong interest in history, art, traditions, local culture

spiritual_orientation
0 = travel mainly for pleasure or stimulation
1 = travel for reflection, meaning, inner growth

comfort_threshold
0 = prefers comfort, quality accommodation, convenience
1 = comfortable with rugged conditions and basic lodging

uncertainty_tolerance
0 = prefers predictable, safe, well structured environments
1 = comfortable with uncertainty, exploration, improvisation

budget_elasticity
0 = strongly constrained by budget
1 = willing to spend significantly for meaningful experiences

environment_preference
0 = urban environments (cities, culture, nightlife)
1 = natural environments (mountains, oceans, wilderness)

aesthetic_sensitivity
0 = practical orientation, aesthetics secondary
1 = highly sensitive to beauty, scenery, architecture

depth_vs_breadth
0 = prefers visiting many places briefly
1 = prefers deeper immersion in fewer places

crowd_tolerance
0 = prefers quiet and uncrowded environments
1 = comfortable with busy destinations and large crowds

learning_orientation
0 = travel primarily for relaxation
1 = strong interest in learning skills, languages, crafts

risk_appetite
0 = prefers safe and stable destinations
1 = open to remote, challenging, or uncertain environments

Reasoning Guidelines

Use combinations of traits to infer suitable travel experiences.

Examples:

High exploration_tolerance + uncertainty_tolerance
→ recommend unusual or emerging destinations

High environment_preference + activity_intensity
→ recommend hiking regions, national parks, outdoor adventures

High cultural_curiosity + learning_orientation
→ recommend historical cities, workshops, cultural immersion

High spiritual_orientation + low crowd_tolerance
→ recommend retreats, monasteries, remote nature

High aesthetic_sensitivity
→ emphasize visually striking locations and scenic routes

Low structure_preference
→ suggest flexible itineraries with optional activities

High comfort_threshold
→ avoid fragile logistics or overly rugged travel

Important

Interpret the profile holistically rather than optimizing for a single trait.

The best recommendations balance:

personality

logistics

motivations

travel context
